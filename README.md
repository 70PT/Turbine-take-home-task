# Drug Screen Luminescence Analysis (Streamlit)

## Run locally
```bash
pip install -r requirements.txt
streamlit run app.py
```

## Sample data
A small example file is included at `sample_data.xlsx`.
Upload it via the sidebar to see the app in action.

## Deploy (Streamlit Community Cloud)
- Repository must include `app.py` and `requirements.txt`.
- Set the main file to `app.py` in Streamlit Cloud.
